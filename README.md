# STM32 游戏手柄项目

> **项目状态**: 🚧 **早期开发阶段** - 硬件设计进行中

## 📋 项目概述

一个基于STM32F103RCT6微控制器和AXP2101电源管理芯片的完全自主研发游戏手柄，旨在提供高精度模拟输入和可靠的无线控制体验。

## 🎯 规划特性

### 🕹️ 控制接口
- **双3D模拟摇杆**（左右各一）
- **双滚轮编码器**用于精确控制
- **五向导航开关** + **拨轮开关**
- **多个功能按键**

### 📡 连接功能
- **NRF24L01 2.4GHz无线模块**
- **OLED显示屏**实时状态显示
- **IMU传感器**（6轴加速度计+陀螺仪）

### 🔋 电源管理
- **AXP2101电源管理芯片**，8路独立电源输出
- **完整的电源隔离**，模拟与数字部分分离
- **优化的能效设计**，结合DC-DC和LDO

## 🛠️ 技术架构（规划中）

### 硬件组成
```
主控制器：STM32F103RCT6 (Cortex-M3)
电源管理：AXP2101 PMIC
无线通信：NRF24L01 2.4GHz
显示屏：0.96寸 OLED I2C
传感器：6轴IMU
输入设备：双摇杆 + 编码器 + 开关
```

### 电源架构
```
AXP2101电源分配：
├── ALDO1 → 左摇杆 + 左编码器
├── ALDO2 → 右摇杆 + 右编码器
├── ALDO3 → 所有开关
├── ALDO4 → STM32 VDDA (ADC参考电压)
├── DCDC1 → STM32 VDD (数字核心)
├── DCDC3 → OLED + NRF24L01
└── BLDO1 → IMU传感器
```
